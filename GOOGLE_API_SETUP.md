# 🚀 Настройка Google Gemini API для ArchVision AI

## Кардинальное улучшение точности!

Google Gemini теперь является **ОСНОВНЫМ** анализатором ArchVision AI. Это решает проблему неточности локальных моделей.

## 📋 Шаги настройки:

### 1. Получите API ключ Google Gemini:
- Перейдите на https://aistudio.google.com/
- Войдите в свой Google аккаунт
- Создайте новый API ключ
- Скопируйте его

### 2. Обновите файл .env:
```
HUGGINGFACE_TOKEN=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
GOOGLE_API_KEY=ваш_реальный_api_ключ_сюда
```

### 3. Установите зависимости:
```bash
pip install google-generativeai>=0.3.0 google-auth>=2.0.0
```

## 🎯 Как это работает:

### Новая архитектура анализа:
1. **Gemini (Приоритет #1)** - Анализирует изображение с максимальной точностью
2. **Wikipedia Integration** - Автоматически ищет информацию о найденных памятниках  
3. **Local Models (Резерв)** - Используются только если Gemini недоступен

### Преимущества Gemini:
- ✅ **Точное распознавание** знаменитых памятников (Arc de Triomphe → Париж)
- ✅ **Реальная геолокация** вместо гадания на моделях
- ✅ **Правильные архитектурные стили** с высокой уверенностью
- ✅ **Wikipedia интеграция** для получения координат и описаний

### Пример улучшения:
```
СТАРАЯ СИСТЕМА:
Триумфальная арка → "Восточная Европа" (Неправильно!)

НОВАЯ СИСТЕМА С GEMINI:
Триумфальная арка → "Париж, Франция, Западная Европа" + Wikipedia данные
```

- Wikipedia интеграция работает автоматически для найденных памятников
